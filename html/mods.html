Server Plugins

Plugin Name Plugin Version Plugin Link letztes Update Plugin Beschreibung

Cenotaph 2.1.2 @ BUKKITdev 07.04.2012 Cenotaph - Dieses Plugin hinterlässt bei Tod eine Kiste oder auch eine große Kiste, je nachdem wie voll euer Inventar ist und beschützt somit eure Gegenstände bis ihr wieder zu euren Todesort zurück gekehrt seid und eure Gegenstände eingesammelt habt.
Craftbook 3.3.0_Dev-9.4.9 @ BUKKITdev 02.09.2012 Craftbook fügt eine Reihe von Gameplay-Elementen aus schaltbaren Brücken, Aufzügen, Toren und Schleusen zu Minecraft hinzu.
Dynmap 0.80 @ BUKKITdev 27.08.2012 Dynmap - Eine Google Maps ähnliche Karte für Minecraft Server, die im Browser angezeigt werden kann und die Positionen der Spieler live überträgt.
Dynmap-WorldGuard 0.30 @ BUKKITdev 28.08.2012 Dynmap-WorldGuard zeigt mit WorldGuard erstellte Regionen auf der Dynmap als extra Layer an.
Essentials 2.9.3 @ BUKKITdev 25.06.2012 Essentials - Eine Sammlung nützlicher Befehle und Funktionen für Bukkit.
Essentials Groupmanager 2.9.3 @ BUKKITdev 25.06.2012 Essentials Group Manager ist ein eigenständiges Berechtigungsplugin für Bukkit.
LWC 4.3.0 @ BUKKITdev 08.08.2012 LWC schützt einzelne Blöcke, das sowohl den Block selbst und die Inhalte der Kisten, Öfen und Spender schützt. Es kann auch alle anderen Blöcke schützen und standardmäßig werden auch Türen (Holz + Eisen) geschützt, Schilder und Falltüren.
MC Jobs 3.1.8 @ BUKKITdev 12.09.2012 MC Jobs ist ein Plugin zum erlernen von Berufen in Minecraft mit welchen man dann Geld verdienen kann, welches man dann in Shops ausgeben kann.
Statistician 2.0.12 @ BUKKITdev 16.08.2012 Statistician sammelt Serverdaten (Uptime, max. Spieleranzahl, Spieler online, etc.) und Spielerdaten (PvE und PvP-Tötungen, Blöcke plaziert/zerstört, Gegenstände eingesammelt/fallen gelassen, max. Distanz zurück gelegt, max. Onlinezeit, etc.) in einer Datenbank.
Vault 1.2.18 @ BUKKITdev 25.08.2012 Vault ist ein Permission, Chat & Economy API Plugin welches die Kompatibilität zwischen den verschiedenen Permission Plugins, Chat Plugins und Economy Plugins herstellt.
WorldBorder 1.5.4 @ BUKKITdev 29.03.2012 WorldBorder erstellt eine Grenze (kreisförmig oder quadratförmig) für jede Welt auf dem Minecraftserver. WorldBorder verbietet das überschreiten dieser Weltgrenzen und zeigt Weltgrenzen zusätzlich auf der Dynmap an.
WorldEdit 5.4.2 @ BUKKITdev 25.08.2012 WorldEdit ist ein In-Game-Map-Editor mit über 100 Funktionen, die den Aufbau und die Befestigung sehr schnell mit drei Arten von Werkzeugen macht:
  • Wähle Formen (Quader, etc.) und ändere die Blöcke innerhalb des Bereiches. Deaktiviere ganze Bereiche mühelos!
  • Binde Pinsel auf deine Werkzeuge und modelliere das Landschaftsbild neu.
  • Verwende Utility-Befehle wie /fixwater und /extinguish.
WorldGuard 5.5.4_Dev-6.4.3 @ BUKKITdev 23.08.2012 WorldGuard ist ein Plugin womit man viele Features von Minecraft verändern kann und man Strukturen oder ganze Gebiete schützen kann.
xAuth 2.0.20 @ BUKKITdev 21.08.2012 xAuth ist ein Plugin zum Schutz für den Server und deren Spieler wenn der Server im Offline-Modus betrieben wird.
ZavAutoMessager 1.9.1 @ BUKKITdev 13.08.2012 ZavAutoMessager ist ein Plugin zum automatischen versenden von Servernachrichten um zum Beispiel wichtige Mitteilungen an alle Spieler mitzuteilen.

ModLoader 1.3.2-MC_1.3.2 Link 19.08.2012 Wird für die meisten Mods gebraucht.
Installation: manuell oder per MAGIC LAUNCHER

Installation: manuell oder per MAGIC LAUNCHER

INVENTORY TWEAKS 1.4.3b-MC_1.3.2 Link 21.08.2012 INVENTORY TWEAKS ermöglicht es das Inventar und Kisten einfach und schnell zu sortieren und Werkzeuge automatisch nach zu füllen. INVENTORY TWEAKS funktioniert sowohl im SSP als auch im SMP.

Installation: manuell oder per MAGIC LAUNCHER
