LWC

Ein Plugin das eure Kisten (und andere Blöcke) schützt. Beim Erstellen, Ändern und Entfernen von Sicherungen muss erst der Befehl eingetippt, anschließend der Block mit der rechten Maustaste angeklickt werden.
Schaut euch auch das Tutorial an um mehr zu erfahren.
 /lwc  Listet alle Befehle im Chat auf.
 /cprivate  Erstellt eine private Sicherung.
 /cpublic  Erstellt eine öffentliche Sicherung.
 /cremove  Entfernt eine Sicherung die man selbst angelegt hat. Als Admin jede Sicherung.
 /cpassword <Passwort>  Erstellt eine Passwort-Sicherung. Jeder der das Passwort kennt hat Zugriff.
 /cunlock <Passwort>  Öffnet eine Passwort-Sicherung. Bleibt bis zum Ausloggen bestehen.
 /cinfo      Nach Befehl klicken um Informationen zu einer Sicherung zu erhalten.
 /climits  Zeigt die Anzahl deiner Sicherungen an. (Aktuell/Maximal)
 /cmodify [Attribut(e)]  Ändert eine Sicherung. Siehe dazu ingame Informationen oder das Tutorial.
 /lwc mode <persist/droptransfer>  persist - sorgt dafür das der letzte LWC-Befehl immer weiter ausgeführt wird.
 zB. um einen Nutzer Zugriff auf mehrere eigene Kisten zu geben.
 droptransfer - unbekannt (nicht benutzen)
 /lwc flag <Name> <on/off>  Ändert bestimmte Eigenschaften einer Sicherung. Die Flag-Namen sind:
 redstone, magnet, exemption, autoclose, allowexplosions
 /lwc menu <basic/advanced>  Ändert das Layout (keine Änderung festgestellt)
 Admin-Commands zu LWC sind hier nicht gelistet.

Cenotaph

Wie Ihr Cenotaph benutzt, erfahrt Ihr im Tutorial.
 /cenlist Gibt euch eine nummerierte Liste eurer Cenotaphs.
 /cenfind <Nummer> Zeigt im Chat die Richtung des Cenotaphs an. Wenn ihr einen Kompass habt zeigt dieser ebenfalls die Richtung an.
 /centime <Nummer> Teilt euch mit wie lange euer Cenotaph noch besteht und wie lange er noch geschützt ist.
 /cenreset Lässt euren Kompass wieder normal funktionieren.
 /cenadmin list Gibt eine Liste aller bestehenden Cenotaphs.
 /cenadmin list <Spieler>
 (Groß-/Kleinschreibung beachten)		 Listet alle Cenotaphs von <Spieler>
 /cenadmin find <Spieler> <Nummer>
 (Groß-/Kleinschreibung beachten)		 Zeigt die Richtung für Cenotaph <Nummer> von <Spieler> an.
 /cenadmin remove <Spieler> <Nummer>
 (Groß-/Kleinschreibung beachten) 		Zerstört Cenotaph <Nummer> von <Spieler> und lässt den Inhalt auf den Boden fallen.
 /cenadmin time <Player> <Nummer>
 (Groß-/Kleinschreibung beachten) 		Zeigt die verbleibende Zeit von <Spieler>'s Cenotaph <Nummer>
an und wie lange der Schutz noch besteht.
 /cenadmin version Zeigt aktuelle Version an und prüft auf Update.
