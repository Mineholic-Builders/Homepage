Cenotaph

Dieses Plugin hinterlässt bei Tod eine Kiste oder auch eine große Kiste, je nachdem wie voll euer Inventar ist und beschützt somit eure Gegenstände bis ihr wieder zu euren Todesort zurück gekehrt seid und eure Gegenstände eingesammelt habt.

Befehl Beschreibung
Admin-Befehle: Bitte Groß- und Kleinschreibung beachten!
Tutorial: Hier erfahrt ihr wie ihr Cenotaph benutzt.
/cenlist Gibt euch eine nummerierte Liste eurer Cenotaphs aus.
/cenfind <Index> Zeigt euch im Chat die Richtung des Cenotaphs an und falls ihr einen Kompass besitzt, zeigt euch dieser ebenfalls die Richtung an.
/centime <Index> Teilt euch mit wie lange euer Cenotaph noch geschützt ist und wie lange dieser noch bestehen bleibt.
/cenreset Lässt euren Kompass wieder in Richtung Spawn ausrichten.
/cenadmin list Gibt eine Liste von allen Spielern mit bestehenden Cenotaphs aus.
/cenadmin list <Spieler> Listet alle vorhandenen Cenotaphs des angegebenen <Spieler> auf.
/cenadmin find <Spieler> <Index> Zeigt die Richtung für den Cenotaph von <Spieler> mit dem <Index> an.
/cenadmin remove <Spieler> <Index> Zerstört den Cenotaph von <Spieler> mit dem <Index> und lässt den gesamten Inhalt des Cenotaphs auf den Boden fallen.
/cenadmin time <Spieler> <Index> Zeigt die verbleibende Zeit des Cenotaph von <Spieler> mit dem <Index> an und wie lange der Schutz des Cenotaph noch bestehen bleibt.

Dynmap

Eine Google Maps ähnliche Karte für Minecraft Server, die im Browser angezeigt werden kann und die Positionen der Spieler live überträgt.

Befehl Beschreibung
/dynmap hide Versteckt einen Spieler auf der Dynmap.
/dynmap show Zeigt einen Spieler auf der Dynmap (wieder) an.
/dynmap sendtoweb <Nachricht> Sendet ausschließlich eine Nachricht zu allen Webbenutzern, welche im normalen Minecraftchat nicht angezeigt wird.

Essentials

Eine Sammlung nützlicher Befehle und Funktionen für Bukkit.

Befehl Beschreibung
Lernt für euch relevante Befehle am besten auswendig oder benutzt /help im Spiel um sie dort anzeigen zu lassen.
/afk Aktiviert und deaktiviert den AFK-Modus.
/compass ; /direction Zeigt die aktuelle Kompassrichtung (z.B. S (180 Grad)) an.
/depth Zeigt die aktuelle Höhe über NN an. In Minecraft liegt der NN in Höhe 64.
/getpos ; /coords ; /position Zeigt die aktuellen Positionsdaten (Welt, X,Y,Z, Drehung, Blickwinkel) an.
/help Listet alle verfügbaren Befehle auf.
/helpop <Nachricht> Sendet eine <Nachricht> an einen verfügbaren Admin.
/news ; /einfo ; /ifo Zeigt Informationen und Neuigkeiten zum Server an.
/itemdb Zeigt den Gegenstandsname, Gegenstands-ID (& Haltbarkeit) eines Gegenstandes in der Hand an.
/list ; /who ; /online Listet aktuelle, maximale und Namen der Spieler auf.
/mail [read|clear|send <Spieler> <Nachricht>] Liest | Löscht | sendet einen <Spieler> eine <Nachricht> (als) Offlinenachricht.
/me <Nachricht> Sendet eine <Nachricht> in 3. Person in den Chat beginnend mit dem eigenen Spielernamen.
/motd Zeigt die MOTD (Message of the Day) an, welche automatisch beim einloggen aufm dem Server angezeigt wird.
/msg <Spieler> <Nachricht> ; /tell <Spieler> <Nachricht> Sendet einen anderen Spieler eine flüster/private Nachricht.
/near <Radius> Zeigt alle Spieler an, welche sich im angegebenen <Radius> sich befinden. Der Standardwert für den beträgt 100m.
/nick <Nickname> Ändert deinen Namen zu den von dir angegebenen <Nickname> für den Chat und die Spielerliste.
/reply <Nachricht> ; /r <Nachricht> Antwortet auf die letzte flüster/private Nachricht.
/realname <Spieler> Zeigt dir den richtigen Spielername an, wenn dieser einen Nicknamen benutzt.
/rules Zeigt dir die Regeln zum Server an.
/seen <Spieler> Zeigt an wann ein Spieler zuletzt online war.
/whois <Spieler> Zeigt von einen <Spieler> das aktuelle Economy-Guthaben, den Realnamen (falls Nickname verwendet wird) und den AFK-Status an.
/clearinventory Erlaubt es dir dein Inventar zu löschen.
/suicide Erlaubt es dir dich selbst zu töten.
/balance ; /bal ; /money Zeigt das aktuelle Economy-Guthaben an.
/balancetop ; /baltop Zeigt die Economy-Rangliste an.
/spawn Teleportiert dich zum Spawnpunkt.
/home <Name> Teleportiert dich zu deinen angegebenen Homepunkt. Falls du nur einen Homepunkt besitzt, dann reicht es aus den Befehl /home zu benutzen.
/sethome <Name> Setzt einen neuen Homepunkt mit dem von dir angegebenen Namen.
/delhome <Name> Löscht einen Homepunkt mit dem von dir angegebenen Namen.
/tpask <Spieler> ; /tpa <Spieler> Sendet eine Teleportanfrage zum angegebenen <Spieler>.
/tpaccept ; /tpyes Akkzeptiert eine Teleportanfrage des anfragenden Spielers und teleportiert ihn an deine Position.
/tpdeny ; /tpno Lehnt eine Teleportanfrage des anfragenden Spielers ab.

LWC

LWC schützt einzelne Blöcke, das sowohl den Block selbst und die Inhalte der Kisten, Öfen und Spender schützt. Es kann auch alle anderen Blöcke schützen und standardmäßig werden auch Türen (Holz + Eisen) geschützt, Schilder und Falltüren.

Befehl Beschreibung
Tutorial: Hier erfahrt ihr wie ihr LWC benutzt.
/lwc Listet alle verfügbaren Befehle von LWC auf.
/cprivate Erstellt eine private Sicherung.
/cpublic Erstellt eine öffentliche Sicherung.
/cpassword <Passwort> Erstellt eine passwortgesicherte Sicherung.
/cunlock <Passwort> Entriegelt eine passwortgesicherte Sicherung.
/cmodify <Spielername> ; /cmodify g:<Gruppenname> Modifiziert eine bestehende Sicherung, fügt Spieler und/oder Gruppen hinzu.
/cmodify -<Spielername> ; /cmodify -g:<Gruppenname> Entfernt Zugriffsrechte eines Spielers und/oder einer Gruppe
/cmodify @<Spielername> ; /cmodify @g:<Gruppenname> Modifiziert eine bestehende Sicherung, fügt Adminrechte für Spieler und/oder Gruppen hinzu. Der Besitzer kann nicht geändert werden.
/cremove Entfernt eine Sicherung. Admins können jede Sicherung damit entfernen.
/cinfo Zeigt Informationen über eine Sicherung an.
/climits Zeigt deine maximale Anzahl an Sicherungen an.
/lwc flag magnet <on/off> Kisten und andere Blöcke mit Inventar können magnetisiert werden, wodurch sie mit einen bestimmten Intervall Gegenstände in einen bestimmten Radius um sich herum aufnehmen.
/lwc flag redstone <on/off> "/lwc flag redstone on" Blockiert das Redstone Signal zum gesicherten Block.
/lwc flag autoclose <on/off> "/lwc flag autoclose off" Schaltet das automatische schließen der Türen ab.
/lwc mode persist Der nächste eigegebene LWC-Befehl bleibt bestehen und kann beliebig oft auf andere Kisten und Blöcke ausgeführt werden.
/lwc remove modes Alle verwendeten Modi werden entfernt.
/lwc admin view <ID> Zeigt das Inventar eines Blocks via ID an.
/lwc admin find <Spieler> Zeigt alle Sicherungen eines Spielers an.
/lwc admin forceowner <Spieler> Ändert den Besitzer einer Sicherung.
/lwc admin remove <ID> Entfernt eine Sicherung via ID, welche man per /cinfo heraus finden kann.
/lwc admin purge <Spieler> Entfernt alle Sicherungen eines Spielers.

MC Jobs

Ein Plugin zum erlernen von Berufen in Minecraft mit welchen man dann Geld verdienen kann, welches man dann in Shops ausgeben kann.

Befehl Beschreibung
/jobs list Listet alle auf dem Server verfügbaren Berufe auf, wieviele Berufe man erlernt hat, wieviele Berufe man maximal erlernen kann und welche Berufe standard Berufe sind.
/jobs join <Jobname> Mit diesen Befehl erlernst du den ausgewählten Beruf. Grau geschriebene Berufe kann man nicht erlernen.
/jobs leave <Jobname> Mit diesen Befehl kündigst du den ausgewählten Beruf. Grau geschriebene Berufe kann man nicht kündigen.
/jobs <Jobname> Zeigt dir detailierte Informationen zum ausgewählten Beruf an. Du erfährst wieviel Erfahrung du für welchen Block erhälst, welchen Level du in diesen Beruf hast und wie hoch dein Lohn je Block ist.
/jobs show <Jobname> Zeigt dir eine Nachricht an für den ausgewählten Beruf für je Aktion die durchgeführt wird mit dem Verdienst an.
/jobs hide <Jobname> Blendet die Nachrichten für den ausgewählten Beruf für je Aktion für den Verdienst wieder aus.
/jadm add <Spieler/Gruppe> <Jobname> Fügt einen Spieler oder eine Gruppe zum ausgewählten Beruf hinzu.
/jadm remove <Spieler/Gruppe> <Jobname> Entfernt einen Spieler oder eine Gruppe aus dem ausgewählten Beruf.
/jadm list <Spieler/Gruppe> <Jobname> Listet den ausgewählten Spieler oder Gruppe auf und zeigt alle Berufe an die er/diese besitzt.

ZavAutoMessager

Ein Plugin zum automatischen versenden von Servernachrichten um zum Beispiel wichtige Mitteilungen an alle Spieler mitzuteilen.

Befehl Beschreibung
durchgestrichene Befehle: Diese Befehle werden später hinzugefügt oder freigeschaltet.
/automessager ignore Aktivieren oder deaktivieren der automatischen Nachrichten im Chat.
/automessager Listet alle verfügbaren Befehle auf.
/automessager broadcast <Nachricht> Sendet über den AutoMessager eine Nachricht an alle Spieler die online sind.
/automessager list <Seite> Zeigt alle vorhandenen Nachrichten des AutoMessager an und deren Nachrichten-ID.
/automessager add <Nachricht> Fügt eine Nachricht zum AutoMessager hinzu. Man kann hierfür sämtlichen Chatformatierungen benutzen.
/automessager remove <Nachricht-ID> Entfernt eine Nachricht vom AutoMessager.
/automessager reload Läd die aktuelle Config des AutoMessager erneut auf dem Server.
